About this Website

This website was created by aspiring developer Beth Knights using personal photos and research on the practice of forest bathing. It is intended as a beginner's guide, with a resources library for users who want to find out more.

The project was first brainstormed as an offshoot of the website mindfulness, which was a project for Code First Girls.

Design & Development

The color scheme was designed using color wheel theory to echo calming themes found in nature. All photos and video were produced by the author in locations including the Lake District, Portadown, and Belfast.

The site was coded using Visual Studio Code and tested on Microsoft Edge and Google Chrome. Please note, it has not yet been adapted for mobile use.
