about this website

this website was created using personal photos and information on the practice of forest bathing

the website was created as part of the DkIT module introtoweb by beth knights




the project was first brainstormed as an offshoot of the website mindfulness which was a project for Code First Girls

the colour scheme was conceptualised using colour wheel theory and also with the intention of echoing calming colours found in the natural environment.




photos were taken by the author in locations including arnside lake district, portadown armagh and belfast antrim

photos were edited using windows photo editor but remain true to original spirit of the image-set




the website was coded using visual studio code, with reference to w3schools and notes provided by DkIT

the website was tested on both microsoft edge and google chrome browsers, but has not yet been adapted for mobile use
