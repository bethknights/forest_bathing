About this Website



This website was created using personal photos and research on the practice of forest bathing.

It is intended as a beginner's guide to the practice, with a resources library in case users want to find out more.


The website was created by aspiring developer Beth Knights as part of the DkIT module 'introtoweb'.

Beth is a student at Dundalk Institute of Technology with a keen interest in mental health and wellbeing.



The project was first brainstormed as an offshoot of the website mindfulness which was itself a project for Code First Girls.

The Forest Bathing colour scheme was conceptualised using colour wheel theory, selecting green shades next to each other on the colour wheel,

and also with the intention of echoing calming colour themes which can be found in the natural environment.

Brighter colours (energetic pink, bright yellow) were added as accent colours.




Website photos were taken by the author in various locations including Arnside Lake District, Portadown Armagh and Belfast Antrim.

Photos were edited using windows photo editor but remain true to original spirit of the image-set, and have been provided with alt descriptions.




The website was coded using visual studio code, with reference to w3schools and notes provided to me as a student of DkIT.

This website has been tested using both microsoft edge and google chrome browsers, but has not yet been especially adapted for mobile use.
